# 学习计划

以下知识点排名不分先后.

- Hello World
- Qt 各种工具的使用
  - Qt Creator, Assistant...
- 界面设计和布局
- 常用 UI 控件与对话框
- 单窗口 UI 与标签页, 可停靠面板等复杂 UI
- signal/slot 机制
- Model/View 框架
- 用户偏好设置的维护
- 多线程
- 网络编程
- 数据库
- 跨平台开发
  - Windows/macOS
  - x86/ARM
- 编译, 构建和发布
  - MSVC/clang/GCC
  - CMake/meson
  - Visual Studio/VS Code/Qt Creator
  - 第三方依赖库的处理
- 综合演练
  - JSON Editor: 以表单方式编辑 JSON 文件
  - Media Player: 音视频播放器
  - QtShark: 类似 Wireshark 的协议分析器
